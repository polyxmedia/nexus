# NEXUS Intelligence Platform

Integrated geopolitical-market intelligence platform. Four primary signal layers, AI-driven thesis generation, regime-aware prediction tracking with Brier scoring, game theory scenarios, Monte Carlo simulation, and trading integration.

**Live:** [nexushq.xyz](https://nexushq.xyz)

## What it does

- **Signal Detection** - Four layers running continuously: geopolitical escalation, market microstructure, open-source intelligence, and systemic risk
- **AI Analysis** - Converged signals synthesised into structured theses with citation trails
- **Prediction Engine** - Falsifiable claims with auto-resolution, Brier scoring, and calibration feedback
- **Trade Lab** - Scenario-weighted position simulator with Monte Carlo, gamma exposure, and regime overlays
- **Shipping Intelligence** - Chokepoint monitoring, dark fleet detection, freight market signals
- **Game Theory** - Payoff matrices, Nash equilibria, escalation trajectories, wartime threshold detection
- **War Room** - Live map with aircraft tracking, OSINT event feeds, and conflict data
- **Knowledge Bank** - Vector embeddings for institutional memory across all intelligence

## Stack

Next.js 15, React 19, TypeScript, Tailwind CSS v4, PostgreSQL + Drizzle ORM, Anthropic Claude, Voyage AI embeddings, Stripe, Leaflet
